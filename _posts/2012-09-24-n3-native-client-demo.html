--- layout: post title: N3 Native Client Demo date: '2012-09-24T18:44:00.003+01:00' author: Andre Weissflog tags: modified_time: '2012-09-24T18:45:39.257+01:00' blogger_id: tag:blogger.com,1999:blog-2948438400037317662.post-9096826387599995826 blogger_orig_url: http://flohofwoe.blogspot.com/2012/09/n3-native-client-demo.html --- After I got the new experimental "Twiggy" pipeline up and running I decided to give Google Native Client another shot, since Twiggy works much better with the "call-on-main-thread" restriction of NaCl. The NaCl port now only differs by a couple of hundred lines of code from the other platform ports which I'm very happy with.

Here's the link to the demo in the Chrome Web Store. You need a recent version of Chrome (at least v.21). So far I only tested this on my MacBook with an nVidia 9400M GPU under OSX and Windows:

https://chrome.google.com/webstore/detail/hjgmfoindkamjgdgjlnjgdfceegejpni

If it doesn't work for you, please leave feedback in the Chrome Web Store, especially which GPU you're using.

Have fun!
